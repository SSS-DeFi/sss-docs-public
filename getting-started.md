# Getting Started

## What it is

This guide helps new users understand how to access and explore SSS.

## Why it matters

A clear onboarding flow reduces friction and improves first-time user understanding.

## How it works

To get started with SSS:

1. Visit the app
2. Connect with supported login methods
3. Review the docs and product pages
4. Start with small test amounts during public test

## Summary

- Read the docs first
- Understand the trust model
- Start carefully during public test
