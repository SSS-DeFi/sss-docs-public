# Public Test

## What it is

SSS is currently in public test.

## Why it matters

Public test means the product is available for exploration and feedback, but users should still treat it as an evolving system.

## How it works

During public test:

- Product behavior may continue to improve
- Interfaces and wording may change
- Users should start with small sizes
- Documentation may be updated frequently

## Summary

- Public test is for validation and iteration
- Start small
- Follow the latest docs and announcements
