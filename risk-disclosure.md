# Risk Disclosure

## What it is

This page summarizes the main risks users should understand before using SSS.

## Why it matters

All crypto products involve technical, operational, and market risks.

## How it works

Users should consider:
- Smart contract and canister risks
- Market volatility
- Product changes during public test
- Execution and settlement timing differences

## Summary

- Use cautiously
- Start with small size
- Read product documentation carefully
